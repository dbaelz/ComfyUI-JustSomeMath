# ComfyUI Just Some Math

Just some math for ComfyUI

# Features
- Multiply Int: This node multiplies two integer values. You provide an integer and a multiplier, and it outputs their product.
- Multiply Float: This node multiplies two floating-point values. You provide a float and a multiplier, and it outputs their product.
